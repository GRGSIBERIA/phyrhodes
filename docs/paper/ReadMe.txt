thesis.tex + tu-civil.sty = 片面印刷用で，これが標準です。

どうしても両面印刷したい場合には，組版の規則に極力合わせてみました。

thesis-b.tex + tu-civil-b.sty = 両面印刷用

両面印刷ということは，書籍と同じになります。したがって，見開きで
右が奇数ページになりますので，章の始まりは常に奇数ページにします。
そのため，場合によっては白紙のページが挟まることになるのです。

ちなみに，thesis.tex と thesis-b.tex の違いは，横向きの表の部分と
使うスタイルファイル・クラスファイルのみです。他の部分は同じですから，
片面印刷用で作った原稿は
 1) \documentclass の report を book に変更する。
 2) \usepackage{tu-civil} を \usepackage{tu-civil-b} に変更する。
だけで，一応は完成します。横向きの図表を使っている場合には，その部分を
サンプルを見ながら変更してください。ページ番号の位置だけの違いです。

pLaTeX のデフォルトでは，空白の偶数ページにはノンブルも柱も出ないように
なっていますが，ちょっとおかしいと思ったので，出るようにしました。
